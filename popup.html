    COLD TURKEY INTEGRATION
    Cold Turkey Blocker is a desktop app that blocks distracting websites and apps. This integration can automatically start blocks during focus sessions, remind you to open it daily, or open it if you've been idle too long.
    You only need to do this once:
    Step 1. Install Cold Turkey Blocker if you haven't already.
    → Download Cold Turkey Blocker (free)
    Step 2. Find your extension ID.
    It's shown right here:
    Step 3. Open the folder where this extension's files are saved.
    This is the folder you loaded as an "unpacked extension" in brave://extensions. It contains files like popup.html, app.js, etc.
    Step 4. Double-click the file called setup-cold-turkey.bat
    A black window will open. When it asks for your extension ID, paste the ID from Step 2 and press Enter.
    ⚠ There are two .bat files — make sure you pick setup-cold-turkey.bat, NOT cold-turkey-host.bat. The other one is internal and won't do anything useful if you run it.
    Step 5. Reload this extension.
    Go to brave://extensions (or chrome://extensions), find "Todo of the Loom", and click the reload button (↻).
    Step 6. Come back here and click TEST CONNECTION above.
    If it says "Connected!" you're all set. Enable the features below.

    DISTRACTION WATCHLIST
    Add websites you tend to waste time on. If you're browsing one, Cold Turkey opens every 10 minutes as a nudge to get back on track. Only active during wake hours (6 AM – 11:40 PM).

    Step 1: Find the extension folder
    The extension lives in a folder on your computer. If you're reading this, you already have it. To install in another browser, you need to know where the folder is. Click the button below to copy the path.
    Step 2: Open extensions in your other browser
    Open the browser you want to install the extension in, and go to its extensions page:
    Brave: brave://extensions
    Chrome: chrome://extensions
    Edge: edge://extensions
    Opera: opera://extensions
    Vivaldi: vivaldi://extensions
    Step 3: Enable Developer Mode
    Look for a Developer mode toggle in the top-right corner of the extensions page. Turn it ON. This lets you load unpacked extensions.
    Step 4: Load the extension
    Click "Load unpacked" (top-left area). A file picker opens. Paste the path you copied in Step 1 into the address bar of the file picker and press Enter. Then click "Select Folder".
    The extension will appear in the list. You're done! Your data syncs automatically between browsers because it's the same folder.
    Step 5: Link to the same profile
    Each browser has its own storage, so by default they don't share data. Link them to the same profile so your distraction watchlist stays in sync across all browsers automatically.

    VOLUME MUTE SCHEDULER
    Automatically mute your computer's volume during set hours. Requires a one-time install of the scheduled task (see button below).
    MUTE AT : UNMUTE AT :
    One-time setup (takes 1 minute):
    1. Open the folder where this extension lives on your computer. It's called Pixel todo lists and it's usually in your user folder (like C:\Users\YourName\Pixel todo lists).

    2. Inside that folder, find the file called setup-volume-host.bat and double-click it. A black terminal window will appear — this is normal.

    3. The window will say "OK" a few times and then close. That's it — the setup is done.

    4. Now reload this extension. Go to brave://extensions (or chrome://extensions) and click the little refresh/reload arrow on the Todo of the Loom card.

    5. Come back to this settings page, scroll down here, and click the TEST MUTE button. If your sound goes quiet for 5 seconds and comes back, you're all set!

    If TEST MUTE doesn't work after reloading, try fully closing and reopening the browser. The volume schedule (auto mute/unmute at your set times) will also be installed automatically.
